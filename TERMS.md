## 3. 외부 AI 에이전트 호출

Maestro 는 사용자 본인이 설치한 외부 CLI (Claude Code / Aider 등) 를 spawn 하는
오케스트레이션 도구입니다. 이 CLI 들의 사용은 **각 서비스의 약관에 따릅니다**:

- Claude Code → Anthropic Terms (https://www.anthropic.com/legal)
- Aider → 설정한 LLM provider 의 약관 (OpenAI 등)
- 기타 Adapter → 해당 서비스 약관

Maestro 가 외부 서비스 호출을 중계 / 추가 비용 부과 / 라이선스 변경 X.
